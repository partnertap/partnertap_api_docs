The PartnerTap API guides walk you through the most common integration scenarios for working with partner ecosystem data programmatically — from authenticating against the API and pulling partner organization reports, to retrieving reference data like the global vertical list. Each guide focuses on a specific endpoint or workflow with the request structure, expected response, and notes on how the data is typically used downstream, so you can go from zero to a working integration without piecing things together from raw reference docs.
